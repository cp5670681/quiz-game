# 学科探索答题游戏需求总览

这份文档是主文档，只负责说明产品总方向、模块边界和子文档目录。详细需求拆到子文档中，后续讨论和实现优先直接修改对应模块文档，而不是继续把所有内容堆回一份总稿。

## 当前产品方向

当前方向已经从“纯答题 + 城建”调整为：

- `主线核心`：常驻世界自由探索的学习 RPG
- `长期成长`：主城家园与学科成长体系
- `学习载体`：战斗、机关、收集、区域推进都通过题目驱动

一句话定义：

> 玩家在学科主题的开放区域中探索、解谜、遇怪答题、带回资源升级家园，并逐步解锁更深区域与更多策略玩法。

当前首版世界范围明确为：

- `主城`
- `数学大陆`
- `英语大陆`

## 已确认的关键决策

### 主线体验

- 主线采用 `常驻世界自由探索`
- 世界由 `多张连通地图` 组成
- 首版地图为 `主城 + 数学大陆 + 英语大陆`
- 大陆按 `知识点区域` 逐步开放
- 每个区域对应 `一个知识点`
- 知识点之间组成 `有前置条件的推进图`
- 遇怪后进入 `独立战斗界面`
- 战斗采用 `答题驱动`
- 玩家是 `单人冒险`
- 每个知识点区域都有 `最终 Boss`
- 击败 Boss 后获得对应 `徽章` 并解锁该区域传送

### 探索与回收

- 区域推进由 `知识点前置关系 + 区域探索` 共同决定
- 主线关键收集物和 `徽章` 拿到即永久记录
- 野外会掉落 `材料`、`金币` 和大陆专属货币
- 普通资源 `带回城才结算`
- 战败是 `软失败`
- 通过 `安全点/传送碑` 回城

### 城镇成长

- 主城内包含 `家园系统`
- 家园支持消耗资源 `建造和升级建筑`
- `金币` 是所有建筑共用的基础货币
- `数学币`、`英语币` 承接对应大陆成长线
- 建筑具体作用后续再细化
- 不以自由摆放和重装饰为首版重点

### 当前重点

- 当前首要验证对象仍然是 `探索答题 RPG 主线`
